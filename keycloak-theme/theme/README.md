# OCTAVE OIDC Theme 


This theme focus on customize for Standard Login Theme page with full responsive:

CSS Framework:

- TailwindCSS

Customized Page:

- Login
- Register
- Forgot/Password
- Update Password
- Setup 2FA OTP
- Login With OTP

## Build Theme into JAR File

Copy `octave` folder into `theme` folder and then run `mvn clean package`
